Four repositories. MIT license. **The code is free. The audit corpus is the product.**

### The stack

| Repo | What it does |
|---|---|
| [`myaifingerprint-python`](https://github.com/MyAIFingerprint/myaifingerprint-python) | Python SDK for the hosted API. 7+ models behind one call. Audits: 1 credit = $0.88. |
| [`Pigeon-Code-Compiler`](https://github.com/MyAIFingerprint/Pigeon-Code-Compiler) | File-naming + module discipline. 200-line caps, polyglot filenames, ether map. |
| [`keystroke-telemetry`](https://github.com/SavageCooPigeonX/keystroke-telemetry) | Cognitive-state classifier. Local. No network egress. |
| [`SavageOS`](https://github.com/SavageCooPigeonX/SavageOS) | 5-layer AI manipulation detector. Browser extension + local server. |
